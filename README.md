# File-Uploader-Downloader

A File Uploader and Downloader app built using MERN stack.

## Steps to run this program -

### Server side -

```js
1. cd server
2. npm install
3. npm start
```

### Client side -

```js
1. cd client
2. npm install
3. npm start
```

The app will run on [localhost:3000](http://localhost:3000)
